# CODE-BLOODED SUBMISSION

To submit your project for evaluation, you must initiate a Pull Request to this repository.
You fail to follow the exact format below, your submission will be rejected instantly.

## Submission

1. `Fork` this repository to create your own copy.
2. In `submission/` Click **Add file** -> **Create new file**.
3. You **MUST** name the file exactly after your Team ID, ending in `.md`.
   _(Example: `CPX-042.md`)_
4. Copy `TEMPLATE.md` content to your own `CPX-XYZ.md`.
5. Edit the content as per your project
6. Commit the new file to your repository.
7. Click **Contribute** -> **Open Pull Request**.
8. Submit the PR.

---

### ⚠️ Warning

- You may only submit one markdown file. Do not upload your actual project code here.
- `CPX-42.md` will be rejected. It must be exactly 3 digits: `CPX-042.md`.

Good luck. Hack hard, hack fair.
